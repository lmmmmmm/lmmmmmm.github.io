# bootstrap table 不能选择每页显示的条数问题

# 1.检查有没有导入相应的css

< link href = "<%=path%>/plugin/grid/paging/bootstrap-table.css" rel = "stylesheet" type = "text/css" /  >
< link href = "<%=path%>/plugin/bootstrap/css/bootstrap.css" rel = "stylesheet" type = "text/css" /  >

# 2.检查有没有导入相应的JS

<script type="text/javascript" src="<%=path%>/plugin/grid/paging/bootstrap-table.min.js"></script>
<script type="text/javascript" src="<%=path%>/plugin/grid/paging/bootstrap-table-zh-CN.min.js"></script>
<script type="text/javascript" src="<%=path%>/plugin/bootstrap/js/bootstrap.js"></script>

# 2.检查jQuery版本是否过低

需要jQuery1.9.1+ 版本。
